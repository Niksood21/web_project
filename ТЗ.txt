Идея для сайта: Простая библиотека книг

Авторы: Никита Родин, Александр Воробьев

Основная цель:
Создать удобный и минималистичный онлайн-ресурс, где пользователи смогут быстро находить, читать и скачивать книги в электронном формате.
Сайт будет ориентирован на простоту использования и доступность, без лишних сложностей.

Основные функции
1. Каталог книг
Книги разбиты по жанрам

2. Фильтры по автору, названию, году издания
Поиск книг по автору, названию

3. Добавление книг пользователями
Зарегистрированные пользователи могут добавлять книги в избранные и делиться ими


Технологии которые будут использованы в нашем проекте:
1. Flask - для работы с сайтом
2. SQLite3 - для работы с базами данных(хранение информации и пользователях)
3. SQLAlchemy - ORM для работы с БД
4. Элементы Bootstrap для написания дизайна сайта
5. Werkzeug.security - Модуль для работы с безопасностью(хеширование паролей)
6. Email_validator - библиотека для проверки корректности email-адресов в Python
